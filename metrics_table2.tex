\section*{Prompt Descriptions}

\textbf{Prompt 1:}
\begin{itemize}
  \item \textit{Initial description:} Briefly describe the new event that just started.
  \item \textit{Validity check:} Given the previous event description: \{previous\_description\}, is this the same event or a new one? Respond ONLY with 'YES' if it's the same or 'NO' if it's a different event.
  \item \textit{Update description:} Briefly describe the new event that just started.
\end{itemize}

\textbf{Prompt 2:}
\begin{itemize}
  \item \textit{Initial description:} Briefly describe the action/event that just started in the current video segment. Focus only on the main actions and the progress of the action, ignore meaningless detail to determine the current event. Be careful on their details. Describe ONLY what is happening in the video segment provided.
  \item \textit{Validity check:} Your goal is to identify if the same action is ongoing in the current video segment compared to the previous one.
Given the previous event description:
\{previous\_description\}
Question: Is the current video segment still the same stage of the same event?
As the last line answer ONLY with 'YES' or 'NO' with no extra characters.
  \item \textit{Update description:} Briefly describe the action/event that just started in the current video segment. Focus only on the main actions and the progress of the action, ignore meaningless detail to determine the current event. Be careful on their details. Describe ONLY what is happening in the video segment provided.
\end{itemize}

\textbf{Prompt 3:}
\begin{itemize}
  \item \textit{Initial description:} Briefly describe the action/event that just started in the current video segment. Focus ONLY on the main character and the main actions. Describe only what's in the video segment
  \item \textit{Validity check:} Your goal is to identify if the same action is ongoing in the current video segment compared to the previous one.

Given the previous event description:
\{previous\_description\}
Question: is this the same event or not?
As the last line answer ONLY with 'YES' or 'NO'.
  \item \textit{Update description:} Briefly describe the action/event that just started in the current video segment. Focus ONLY on the main character and the main actions. Describe only what's in the video segment
\end{itemize}

\textbf{Prompt 4:}
\begin{itemize}
  \item \textit{Initial description:} Provide a concise description of the main action in the current video segment. Focus only on the movements.
  \item \textit{Validity check:} Given the previous event description: \{previous\_description\}, is this the same event? Respond ONLY with 'YES' or 'NO'.
  \item \textit{Update description:} Provide a concise description of the main action in the current video segment. Focus only on the movements.
\end{itemize}

\textbf{Prompt 5:}
\begin{itemize}
  \item \textit{Initial description:} Provide a concise description of the main action in the current video segment. Focus on the substeps of the action.
  \item \textit{Validity check:} Given the previous event description: \{previous\_description\}, is this the same event? Respond ONLY with 'YES' or 'NO'.
  \item \textit{Update description:} Provide a concise description of the main action in the current video segment. Focus on the substeps of the action.
\end{itemize}
